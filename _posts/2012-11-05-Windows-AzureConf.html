In just over a week on November 14th, Microsoft will be hosting a free online event called Windows AzureConf.  This conference is unique in that all of the content will be driven by community members that use Windows Azure in their day-to-day jobs.  Instead of listening to me tell you about what Windows Azure can do, you get to listen to your fellow developers talk about how they are actively using it!

The event starts at 8:00 AM PST and registration takes only a minute at http://www.windowsazureconf.net.  Kicking off the event is a keynote by Scott Guthrie.  After that it is all community driven sessions on many different areas of Windows Azure.  Topics covered include Mobile Services, Websites, security, embedded devices, and more.  You can view the full session list here.  You can also read about the community speakers that will be presenting here.  This conference will be jam packed with great information that will show you just some of the things you can do with Windows Azure. 

If you haven’t played with Windows Azure yet, you’re going to want to after you watch this conference.  I highly recommend signing up for a free Windows Azure account prior to attending so you’ll be able to start using it right away.  If you have any questions about Windows Azure after the conference, don’t hesitate to get in touch with me.
